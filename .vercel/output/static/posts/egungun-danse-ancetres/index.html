Egungun : Pourquoi les morts reviennent-ils danser parmi nous ?

Quand un touriste voit un Egungun pour la première fois, son premier réflexe est souvent le recul. Le masque est imposant, les couleurs sont vives, la foule est agitée.

Mais si on vous donnait les “lunettes” pour voir la réalité spirituelle, vous ne reculeriez pas de peur. Vous vous inclineriez de respect.

Le but d’Héritage Vodun est de briser les mythes. Aujourd’hui, parlons de ce qui se cache vraiment sous ces étoffes sacrées : l’Amour.

Une visite de courtoisie de l’Au-delà

Le mot “Revenant” fait peur en français car il rappelle les films d’horreur. Mais dans la tradition, l’Egungun est littéralement un Ancêtre. C’est le grand-père, l’oncle, le patriarche fondateur qui a obtenu la permission exceptionnelle de revenir sur Terre le temps d’une fête.

Imaginez que vous puissiez revoir un être cher disparu pour une journée. Auriez-vous peur ? Non. Vous seriez ému. C’est exactement ce qui se passe à Ouidah. Les familles chantent les louanges de l’Egungun (“Oriki”) pour lui dire : “Nous ne t’avons pas oublié. Regarde comme tes enfants ont grandi.”

Pourquoi cette distance physique alors ?

C’est la question qui revient toujours : “Si c’est de l’amour, pourquoi les gardiens (Mariwo) nous empêchent-ils de les toucher ?”

La réponse est énergétique, pas maléfique.

Dans la cosmogonie Vodun, le monde des morts vibre à une fréquence beaucoup plus haute que le nôtre. L’Egungun est un réacteur nucléaire spirituel. Il est chargé d’une force brute (“Ashe”).

  1. Le “Ne pas toucher” est une protection : Comme on ne touche pas un câble électrique dénudé, on ne touche pas un Egungun. Ce n’est pas parce qu’il veut vous faire du mal, c’est parce que son “voltage” est incompatible avec votre corps humain ordinaire.
  2. Le respect du Sacré : Garder ses distances, c’est aussi reconnaître que certaines choses sont plus grandes que nous. C’est une forme d’humilité.

Comment interagir avec eux ?

Ce 10 Janvier, si vous croisez un Egungun :

  • Souriez. Vous assistez à un miracle : la frontière entre la vie et la mort a été abolie.
  • Observez le vent. On dit que la brise qui soulève les tissus est le souffle de l’ancêtre.
  • Recevez. Même à distance, la bénédiction d’un Egungun est puissante. Elle apporte protection et prospérité.

N’ayez plus peur des masques. Derrière le tissu, il n’y a pas le vide. Il y a la racine de votre histoire.
